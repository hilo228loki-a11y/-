Drone Audio Detection System
Система глубокого обучения для автоматического обнаружения дронов по аудиосигналам.

О проекте
Данный проект предназначен для классификации аудиозаписей на два класса: «Дрон» и «Фоновый шум» (включая hard negatives: дрели, газонокосилки, вертолеты).

Система построена на базе сверточной нейронной сети (CNN), обученной на спектрограммах звука.

Характеристики
Датасет: 180,326 аудиоклипов.

Архитектура: CNN (Convolutional Neural Network).

Точность (Accuracy): 99.07%.

Частота дискретизации: 16,000 Гц.

Длительность клипа: 4.0 секунды.

Как использовать
1. Окружение
Убедитесь, что установлены необходимые библиотеки:

Bash
pip install torch torchvision torchaudio pandas librosa scikit-learn
2. Тестирование модели
Для проверки качества модели на тестовых данных выполните:

Bash
python final_check.py
3. Обучение
Если вы хотите дообучить модель на новых данных, используйте:

Bash
python train_fixed.py
Советы по заполнению:
Если вы планируете выложить проект на GitHub, этот файл будет первым, что увидят люди.

Вы можете добавить в него раздел «Как это работает», кратко описав, что звук переводится в мел-спектрограмму (картинку), а затем CNN ищет на этой картинке паттерны, характерные для работы пропеллеров дрона.
